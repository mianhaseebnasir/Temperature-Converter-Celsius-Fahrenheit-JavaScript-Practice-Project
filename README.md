<h1>Temperature-Converter-Celsius-Fahrenheit-JavaScript-Practice-Project</h1>
<h1>1️⃣ Introduction :</h1>
This project is a basic JavaScript practice application that demonstrates how to perform temperature conversions between Celsius and Fahrenheit.
The project is designed for beginners who want to understand how JavaScript interacts with HTML forms and DOM elements.<br>
The application takes user input, performs the temperature conversion using JavaScript formulas, and displays the result instantly on the webpage.<br>
<h1>2️⃣ Project Objectives :</h1>
The main objectives of this project are: <br>
<li>To understand JavaScript functions</li>
<li>To learn DOM manipulation</li>
<li>To practice form handling in JavaScript</li>
<li>To implement temperature conversion formulas</li>
<li>To create a simple interactive web application</li>
<h1>3️⃣ Features :</h1>
The application provides the following features:<br>
<li>Convert Celsius to Fahrenheit</li>
<li>Convert Fahrenheit to Celsius</li>
<li>Simple and clean user interface</li>
<li>Instant calculation without page reload</li>
<li>Input validation through number fields</li>
<li>Result displayed automatically</li>
<h1>4️⃣ Technologies Used :</h1>
This project is developed using the following technologies:<br>
<li>HTML5 – For webpage structure</li>
<li>JavaScript (Vanilla JS) – For calculations and interactivity</li>
<li>DOM Manipulation – To access input fields and display results</li>
<h1>5️⃣ Temperature Conversion Formulas :</h1>
<b>Celsius to Fahrenheit :</b><br>
Fahrenheit = (Celsius × 9/5) + 32<br>
<b>Fahrenheit to Celsius :</b><br>
Celsius = (Fahrenheit − 32) × 5/9<br>
These formulas are implemented in JavaScript functions to perform automatic calculations.<br>
<h1>6️⃣ How the Application Works :</h1>
<b>1. </b>The user enters a temperature value in the input field.<br>
<b>2. </b>The user clicks the Convert button.<br>
<b>3. </b>JavaScript reads the value using DOM methods.<br>
<b>4. </b>The appropriate temperature conversion formula is applied.<br>
<b>5. </b>The calculated result is displayed in the result field.<br>
